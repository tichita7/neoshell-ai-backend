# NeoShell AI Backend

FastAPI backend powering NeoShell AI — an AI-based shell command generation platform using Groq LLM integration.

This backend processes natural language prompts and returns valid operating system shell commands through a REST API.

---

## Features

- AI-powered shell command generation
- FastAPI REST API
- Groq LLM integration
- Natural language to terminal command conversion
- Secure environment variable handling
- Lightweight backend architecture
- Frontend integration ready

---

## Tech Stack

- Python
- FastAPI
- Groq API
- Uvicorn
- dotenv
